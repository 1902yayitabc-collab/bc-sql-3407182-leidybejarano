# Proyecto Semana 01 — Empresa de Exportación

## 📌 Descripción

Este proyecto modela una empresa de exportación utilizando una base de datos relacional en SQLite.

## 🗂️ Entidades principales

* products (productos)
* clients (clientes)
* shipments (envíos)

## 🛠️ Estructura

* `starter/proyecto.sql`: contiene la creación de tablas, inserts y consultas.

## ▶️ Cómo ejecutar

```bash
sqlite3 mi_dominio.db < starter/proyecto.sql
sqlite3 mi_dominio.db
```

## 📊 Funcionalidades

* Creación de tablas relacionales
* Inserción de datos realistas
* Consultas SELECT básicas
